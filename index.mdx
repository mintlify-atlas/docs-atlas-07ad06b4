## What is fluXis?

fluXis is a free-to-play rhythm game available on [Steam](https://store.steampowered.com/app/3440100/fluXis/) and as open-source software on [GitHub](https://github.com/InventiveRhythm/fluXis). It supports 1–10+ key modes, a rich built-in map editor, a Lua scripting engine for storyboards, and online multiplayer with leaderboards.
